RangeSet

new RangeSet(settings, _list, type)

Source:
  • Types/RangeSet.js, line 2
A RangeSet works much like a Range. But they are able to contain 'holes' within the range. A RangeSet is iterable using a for..of loop, it loops through the Ranges that make up the set.
Parameters:
Name Type Description
settings object The settings of the range.
_list array An Array with all the Ranges to include in the RangeSet
type function The constructor function for a Range of the correct type
