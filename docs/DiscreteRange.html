DiscreteRange

new DiscreteRange(settings)

Source:

Discrete ranges are a subset of ranges that work on discrete types. This includes `int` and `datetime.date`. Discrete ranges are iterable. Using: let x of IntRange.
Parameters:
Name Type Description
