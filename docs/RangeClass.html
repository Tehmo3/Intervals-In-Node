Abstract base class of all ranges. Ranges are strict about types. This means that both `lower` and `upper` must be of the given class or subclass, or `null`
Parameters:
Name Type Description

Properties

 null The lower end of the range
upper scalar <optional>
 null The upper end of the range
lowerInc boolean <optional>
 true ``true`` if lower end should be included in range.
upperInc boolean <optional>
 false ``true`` if upper end should be included in range.
type scalar The type of the range.
